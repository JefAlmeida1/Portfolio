Projeto Livro

Este projeto foi feito durante atividade da Digital Innovation One durante o Bootcamp HTML Web Developer, ele foi criado com a intensão de aprender a fazer a pagina da Netflix, aqui eu peguei a base dele e adicionei algumas alterações, também fiz mais uma pagina para formulário simulando o cadastro de usuário no site.

Detalhe: A pagina do formulário deve ser preenchido todos os campos e depois clicar em cadastrar, os dados ficaram salvo na maquina local, após simular o cadastro a pagina recarrega para limpar o conteúdo.

O que foi usado: HTML5, CSS3, JavaScript e Nodejs.

Software para o desenvolvimento: Visual Studio Code.

OBS: O projeto não está totalmente completo, existem alguns detalhes para resolver, mas por enquanto vou deixar ele aqui para fazer o download para ver os códigos.
